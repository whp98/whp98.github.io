# flatpak安装常用软件


## 内容

```bash
#微信
flatpak install com.tencent.WeChat
##QQ 
flatpak install com.qq.QQ
##钉钉
flatpak install com.dingtalk.DingTalk
##电报
flatpak install org.telegram.desktop
##blender
flatpak install org.blender.Blender
##brave浏览器
flatpak install com.brave.Browser
## 谷歌浏览器
flatpak install com.google.Chrome
## Remmina远程桌面
flatpak install org.remmina.Remmina
## rustdesk远程
flatpak install com.rustdesk.RustDesk
## vscode
flatpak install com.visualstudio.code
## Redis Insight
flatpak install com.redis.RedisInsight
## jetbrains
flatpak install com.jetbrains.IntelliJ-IDEA-Ultimate
flatpak install com.jetbrains.DataGrip
flatpak install com.jetbrains.CLion
flatpak install com.jetbrains.GoLand
flatpak install com.jetbrains.RustRover
# Rhythmbox3 音乐播放器
flatpak install org.gnome.Rhythmbox3
# 落雪音乐
flatpak install cn.toside.lxmusic.lx-music-desktop
# Apifox接口测试
flatpak install com.apifox.Apifox
# android studio
flatpak install com.google.AndroidStudio
# DBeaver
flatpak install io.dbeaver.DBeaverCommunity
```
