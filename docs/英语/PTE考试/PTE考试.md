# PTE考试

PTE学术英语考试（Pearson Test of English Academic）
特点是使用ai电脑评分，避免考官主观评分带来的不公平。
考试费用275美元，考试地点很多，由于使用电脑评分，可以通过类似的评分系统进行练习。
