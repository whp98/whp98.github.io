# Chapter 8: 猎手的追击 (The Hunter's Pursuit)

失败的介入，没有给他们带来片刻的安宁，反而像是一声巨响，彻底惊动了整个宇宙的“秩序维护者”。

在各自被时间洪流囚禁的牢笼中，林晚和顾清辞都感受到了那股寒意——它不再是遥远的、抽象的法则，而是化为了有形、有目标的“追捕”。

在顾清辞的古代世界，时间守卫者的代表，终于找到了他的踪迹。他们不再需要通过玉佩这种低效的媒介来“观测”，而是直接在时间轴上，将他锁定了。那些先前出现的“影子”，现在如同猎犬一般，开始收紧包围圈。

顾清辞的世界，在外部的威胁下，陷入了极度的混乱。那些原本被他依靠的权力体系，开始变得异常脆弱和多变。那些本该牢不可破的“既定事实”，像沙堡一样，随时可能被那股修正力的一记重击而坍塌。他不得不暂时放弃那些与他命运紧密捆绑的责任，转而将全部精力投入到“躲避”和“生存”上。他成了被追捕的，一个打破了时间规则的“叛逆者”。

而在林晚的现代世界，情况更为隐蔽，但也更为可怕。她发现，那些时间异常的“裂痕”，正在沿着她所在的现代文明的“根基”蔓延。那些科学理论，那些历史学界公认的定律，开始出现细微的、逻辑上的“不通顺”。她不是在修复文物，而是在一个不断自我修改的“幻境”中挣扎。

“我们必须找到一个共同点，”林晚通过玉佩，几乎是在绝望中向顾清辞传递信息，“我们被锁死了，但锁链本身，也是可以被分析的结构。时间守卫者，它们需要一个‘完美’的、逻辑自洽的系统来维持秩序。我们，就是系统中的一个**致命Bug**。”

这种分离的状态，带来了一种极其折磨人的痛苦。他们无法拥抱，无法交谈，所有的交流都必须通过那枚承载着他们爱意的玉佩——一个随时可能被“秩序”发现、并进行彻底“格式化”的风险源。

追捕的速度，越来越快。

时间守卫者的目的，已经不再仅仅是“修正历史”，而是要**消除一切异类**。他们要做的，是抹去林晚存在过的每一个历史痕迹，抹去顾清辞所拥有的一切命运轨迹。

当他们意识到，自己已经被置于一个“生存游戏”之中时，绝望开始蔓延。

但正是在这绝境中，那份跨越时空的爱，被重新淬炼，变得无比坚硬。它不再是飘渺的诗歌，而是最现实、最急迫的战术。

林晚和顾清辞明白了，要对抗一个宏大的、冰冷的系统，需要的不是蛮力，而是**完美的、具有颠覆性的“不完美”**。他们必须找到一个方法，让他们的存在，从一个“悖论”的错误代码，变成一个系统本身无法预测的、拥有自我存在的“例外”。

这不再是简单的逃亡，这是在宇宙尺度的维度战争。而这场战争，他们的唯一武器，唯有那份不顾一切、超越规则的爱。
