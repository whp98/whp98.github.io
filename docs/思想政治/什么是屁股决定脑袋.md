# 什么是屁股决定脑袋

“爸爸，什么是屁股决定脑袋？”

“屁股决定脑袋是指现在当官的坐在官的位置，只想着自己如何弄权、弄钱、弄女人。”

“那什么是位置决定思考呢？”

“位置决定思考是指你处在什么位置上，就会主动考虑什么问题。比如爸爸是你的爸爸，思考的就是如何挣钱养家，如何把你培养成有理想有道理有文化有纪律的新时代青年……小明，你的理想是什么？”

“我想长大之后拥有金钱和美女。”

“什么素质！臭小子王八蛋！你能不能有出息一点？平时我怎么教你的？再说一遍，你的理想是什么？”

“呜呜呜，我一定要好好学习，长大以后追求自己的事业和爱情。”

“乖，这才是我的好孩子。”
