# Cycle Timer (循环计时器)

一款使用 Jetpack Compose 构建的现代、可定制化循环计时应用。适用于 HIIT、Tabata、瑜伽以及各类循环训练。

## 功能特性

- **高度可定制的训练方案**：创建并编辑符合您需求的计时预设。
    - 自定义“准备”、“运动”和“休息”阶段的时长。
    - 配置“循环次数”和“组数”。
    - 支持组间休息（Long Rest）。
    - 为不同的训练方案分配颜色，方便快速识别。
- **动态计时界面**：
    - 针对不同阶段（准备、运动、休息、组间休息）提供直观的颜色反馈。
    - 阶段间平滑过渡。
    - 直观的控制操作（开始、暂停、完成）。
- **音频提醒与个性化**：
    - 开启/关闭阶段切换提示音。
    - 可选最后 3 秒倒计时嘀嗒声。
    - 自定义运动开始或休息开始的音频提醒。
- **进度追踪**：
    - 查看统计数据，包括完成的总训练次数和累计时长。
    - 追踪最近的训练记录。
- **现代 UI 设计**：完全使用 Jetpack Compose 开发，提供响应迅速且流畅的用户体验。
